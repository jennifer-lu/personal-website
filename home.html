    TexScan

     

    Document scanner that converts handwritten mathematical notation into LaTeX using neural network-based OCR.




    • Documents can be scanned physically or uploaded as photos to website
    • Flask server preprocesses images and performs OCR
    • Optical character recognition implemented via a ResNet model trained using TensorFlow in conjunction with OpenCV and PyTesseract
    • Converted files available for download on website and emailed to user via a Python script

    Python • Keras • Tensorflow • HTML • JavaScript • Axios • React
